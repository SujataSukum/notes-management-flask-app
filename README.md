# 📝 Notes Management Web App

A full-stack Notes Management Web Application built using **Python (Flask)**, **SQLite**, and **Tailwind CSS**.  
This application allows users to create, manage, search, and organize notes efficiently with a modern UI.

---

## 🚀 Features

- ➕ Add, update, and delete notes (CRUD operations)
- 🔍 Real-time search functionality
- 🕒 Timestamp for each note
- ⚠️ Delete confirmation popup
- 🎨 Modern and responsive UI using Tailwind CSS
- 💾 Persistent storage using SQLite database

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Version Control:** Git, GitHub

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/SujataSukum/notes-management-flask-app.git

2 . Navigate to the project folder:
cd Notes_api_flask_project

3. Install dependencies:
pip install flask

4. Run the application:
python app.py

6. Open in browser:
http://127.0.0.1:5000
